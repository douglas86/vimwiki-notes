
= coc-markdownlint =
markdownlint for Vim/Neovim, works as coc.nvim extension.

== online resources ==
https://github.com/fannheyward/coc-markdownlint

== installing ==
* installing using coc.nvim
	:CocInstall coc-markdownlint
* installing using marketplace
	- run the following command
	- :CocList marketplace
	- once on the one that you want hit enter
